# 🎮 Simon Game – JavaScript Memory Game

> 🚀 A classic Simon Says memory game built using JavaScript with interactive gameplay.

---

## 📌 Overview

Simon Game is a **memory-based game** where players must repeat an increasing sequence of colors.
The game tests memory skills and becomes progressively harder.

---

## 🎯 Features

* 🎨 Interactive color buttons
* 🔊 Sound effects
* 📈 Increasing difficulty levels
* 🔁 Restart functionality

---

## 🧠 Key Concepts

* DOM Manipulation
* Event Handling
* Game Logic Implementation
* Sequence Tracking

---

## ⚙️ Tech Stack

* **HTML**
* **CSS**
* **JavaScript (Vanilla)**

---


## ⚡ How It Works

1. Game starts on key press
2. System generates a color sequence
3. Player repeats the sequence
4. Sequence grows after each correct attempt
5. Game ends on wrong input
